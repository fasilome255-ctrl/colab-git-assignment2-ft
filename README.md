# Titanic Data Analysis

## Project Overview
Analysis of Titanic passenger data to identify
survival patterns using Python and Pandas.

## Key Findings
- Female passengers had 74% survival rate
- First class passengers survived at 63%
- Family size affected survival chances

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## How to Run
1. Clone this repo
2. Install requirements: pip install -r requirements.txt
3. Run: jupyter notebook Lesson_5_Assignment.ipynb
